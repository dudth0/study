# 배너 베리에이션 - 프론트엔드 UI

글자수 제한에 맞춰 홍보카피를 자동으로 요약/변환하는 웹 인터페이스

## 파일 구조

```
docs/frontend/
├── index.html          # 메인 HTML 페이지
├── css/
│   └── style.css       # 스타일시트
├── js/
│   └── app.js          # JavaScript 로직
└── README.md           # 이 문서
```

## 기능

### 1. 입력 폼
- ✏️ **제목**: 프로모션 제목 입력
- 🏷️ **카테고리**: 프로모션, 이벤트, 신규출시, 공지사항 선택
- 📝 **원본 카피**: 요약할 텍스트 입력 (최대 5000자)
- 📊 **실시간 글자수 카운터**: 입력된 글자수 표시

### 2. 글자수 제한 선택
- ☑️ 10자, 20자, 50자, 100자 중 다중 선택
- 각 제한에 맞는 베리에이션 자동 생성

### 3. 베리에이션 생성
- 🤖 자동 텍스트 요약 (더미 알고리즘)
- ⏱️ 로딩 애니메이션 (2초)
- 📋 카드 형태의 결과 표시

### 4. 편집 기능
- ✏️ **수정 버튼**: 인라인 편집 모드
- 💾 **저장**: 수정된 내용 저장
- ❌ **취소**: 편집 취소 및 원본 복원
- 🏷️ **상태 표시**: 자동 생성됨 / 수동 편집됨

### 5. 초기화
- 🗑️ 모든 입력 및 결과 초기화

## 실행 방법

## 화면 구성

### 헤더
- 보라색 그라데이션 배경
- 제목 및 설명

### 입력 섹션
- 회색 배경의 폼
- 제목, 카테고리, 원본 카피 입력
- 글자수 제한 체크박스
- 베리에이션 생성 버튼

### 결과 섹션
- 베리에이션 카드 리스트
- 각 카드:
  - 글자수 제한 배지 (보라색 그라데이션)
  - 상태 배지 (자동 생성 / 수동 편집)
  - 요약된 텍스트
  - 수정 버튼

### 로딩
- 회전하는 스피너
- "베리에이션 생성 중..." 메시지

## 디자인 특징

### 색상 팔레트
- **Primary**: #667eea → #764ba2 (보라색 그라데이션)
- **Background**: #f8f9fa (밝은 회색)
- **Success**: #4caf50 (초록색)
- **Warning**: #f57c00 (주황색)
- **Error**: #f44336 (빨간색)

### 애니메이션
- **fadeIn**: 페이드 인 효과
- **slideUp**: 아래에서 위로 슬라이드
- **spin**: 로딩 스피너 회전

### 반응형 디자인
- 768px 이하: 모바일 레이아웃
  - 세로 방향 체크박스
  - 세로 방향 결과 헤더
  - 작은 폰트 크기

## 텍스트 요약 알고리즘 (더미)

현재는 간단한 클라이언트 사이드 알고리즘 사용:

1. 문장 단위로 텍스트 분리 (`.` `!` `?` 기준)
2. 글자수 제한에 맞게 문장 선택
3. 초과 시 `...` 추가
4. 결과가 비어있으면 앞부분 잘라내기

**주의**: 실제 서비스에서는 AI API (OpenAI, Claude 등) 사용 필요
